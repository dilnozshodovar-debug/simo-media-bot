# SIMO.MEDIA — Telegram Bot

Боти профессионалӣ бо прайс-лист, интихоби пакет ва маълумот дар бораи студия.

## Имконоти бот

- `/start` → менюи асосӣ (тугмаҳо)
- 💰 Прайс-лист → 3 пакет (STANDARD / VIP / VIP PREMIUM)
- Пахш кардани ҳар пакет → тавсифи пурра + тугмаи "Ин пакетро интихоб мекунам"
- Интихоби пакет → мизоҷ хабардор мешавад, ва (агар танзим шуда бошад) шумо ҳамчун админ хабари фармоиш мегиред
- ✨ Чаро маҳз мо?
- ℹ️ Дар бораи SIMO.MEDIA
- 📞 Тамос

## Насб ва иҷрои бот

### 1. Токен гиред
Дар Telegram ба **@BotFather** нависед → `/newbot` → номи ботро диҳед (масалан `SIMO_MEDIA_bot`) → токенро нигоҳ доред.

### 2. ID-и худро гиред (агар мехоҳед хабари фармоишҳоро гиред)
Ба **@userinfobot** нависед — вай ID-и шуморо мефиристад.

### 3. Файлҳоро танзим кунед
Дар `bot.py`:
```python
BOT_TOKEN = "ТОКЕНИ_ШУМО_ИНҶО"
ADMIN_IDS = [123456789]   # ID-и худро гузоред
CONTACT_PHONE = "+992 XX XXX XXXX"
CONTACT_USERNAME = "@username_и_шумо"
```

Ё бе тағйири код, тавассути муҳити система (安全тар):
```bash
export BOT_TOKEN="ТОКЕНИ_ШУМО_ИНҶО"
```

### 4. Насби китобхонаҳо
```bash
pip install -r requirements.txt
```

### 5. Иҷрои бот
```bash
python bot.py
```

Агар ҳама дуруст бошад, дар терминал мебинед:
```
🤖 SIMO.MEDIA bot кор карда истодааст...
```

Ҳоло дар Telegram ботро ёфта, `/start`-ро занед.

## Барои нигоҳ доштани бот 24/7

Барои он ки бот доимо кор кунад (на танҳо дар компютери шумо), онро дар сервер ҷойгир кунед:
- **VPS** (масалан оддитарин — DigitalOcean, Hetzner, ё сервери маҳаллӣ)
- Хидматҳои ройгон/арзон: **Railway.app**, **Render.com**, **PythonAnywhere**
- Дар сервер бо `screen` ё `systemd` бот метавонад доимо кор кунад
